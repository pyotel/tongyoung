# 시간 외 근무 계산기

KETI 시간외근무 신청서 작성을 위한 웹 기반 계산기

## 기능

- 시간외근무 시간 자동 계산 (연장, 휴일초과, 야간)
- 신청자 정보 입력 (신청부서, 직급, 성명, 생년월일)
- 공휴일/대체휴일 자동 감지
- template.ods 기반 ODS 파일 다운로드 (서식 완벽 보존)
- 24시 스타일 시간 입력/표시

## 계산 규칙

- 평일 퇴근: 17시 30분
- 토요일 퇴근: 12시 30분
- 야간: 22시 00분 ~ 06시 00분
- 공휴일/일요일: 8시간까지 연장, 초과분 휴일초과

## 실행

```bash
npm install
npm run dev
```
